Maximus Hair Studio

This was our final project at the BCIT Technical Web Designer program. We collaborated in a team of four to design and develop a responsive website for a local established hair salon under a new ownership. Liaised with the client to identify their needs and requirements and created wireframes and mockups before starting the coding stage. Built the website using HTML, CSS, SASS, PHP, JavaScript, WordPress and GitHub.
